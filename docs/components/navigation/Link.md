## Variants

```js
<Link>Simple link</Link><br />
<Link variant='primary'>Primary link</Link><br />
<Link variant='secondary'>Secondary link</Link><br />
<Link variant='success'>Success link</Link><br />
<Link variant='danger'>Danger link</Link><br />
<Link variant='warning'>Warning link</Link><br />
<Link variant='info'>Info link</Link><br />
<Link variant='light'>Light link</Link><br />
<Link variant='dark'>Dark link</Link><br />
```

## Link with a handler

```js
<Link onClick={ () => alert('link clicked') }>Clickable link</Link>
```
