# Default

```js
<Indicator variant='success'>Upload complete.</Indicator>
<Indicator variant='primary'>Network connected.</Indicator>
<Indicator variant='secondary'>Not connected to server.</Indicator>
<Indicator variant='danger'>Upload failed.</Indicator>
<Indicator variant='warning'>Connection slow.</Indicator>
<Indicator variant='info'>No new messages.</Indicator>
<Indicator variant='light'>Dark mode indicator.</Indicator>
<Indicator variant='dark'>Feature unavailable.</Indicator>
```

# Using label

```js
<Indicator variant='success' label='Upload complete.' />
<Indicator variant='primary' label='Network connected.' />
<Indicator variant='secondary' label='Not connected to server.' />
<Indicator variant='danger' label='Upload failed.' />
<Indicator variant='warning' label='Connection slow.' />
<Indicator variant='info' label='No new messages.' />
<Indicator variant='light' label='Dark mode indicator.' />
<Indicator variant='dark' label='Feature unavailable.' />
```
