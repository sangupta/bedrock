# Folder

```js
<AssetIcon asset={{ isFolder: true }} />
```

# Using MIME types

```js
<AssetIcon asset={{ mimeType: 'image/jpeg' }} />
<AssetIcon asset={{ mimeType: 'video/mp4' }} />
<AssetIcon asset={{ mimeType: 'text/plain' }} />
<AssetIcon asset={{ mimeType: 'text/csv' }} />
```
