## Default ColorInput

```js
<ColorInput name='chosenColor' className='col-md-1' />
```

## With pre-selected value

```js
<ColorInput name='chosenColor' className='col-md-1' value='#758e7f' />
```

## Event handling

```js
<ColorInput className='col-md-1' value='#cf21d0' onChange={ e => console.log(e) } />
```
