### Default typography

Useful to setup headings.

```js
<Heading size={1}>h1. Bedrock heading</Heading>
<Heading size={2}>h2. Bedrock heading</Heading>
<Heading size={3}>h3. Bedrock heading</Heading>
<Heading size={4}>h4. Bedrock heading</Heading>
<Heading size={5}>h5. Bedrock heading</Heading>
<Heading size={6}>h6. Bedrock heading</Heading>
```

### Display Mode

```js
<Heading displayMode={true} size={1}>h1. Bedrock heading</Heading>
<Heading displayMode={true} size={2}>h2. Bedrock heading</Heading>
<Heading displayMode={true} size={3}>h3. Bedrock heading</Heading>
<Heading displayMode={true} size={4}>h4. Bedrock heading</Heading>
<Heading displayMode={true} size={5}>h5. Bedrock heading</Heading>
<Heading displayMode={true} size={6}>h6. Bedrock heading</Heading>
```
